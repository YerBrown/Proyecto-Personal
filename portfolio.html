Videojuego Survival con coleccion de criaturas y gestion de base - Unity

Este proyecto comenzó con la idea de juntar los juegos survival con los de colección de criaturas puesto que soy muy fan de los dos géneros. Es el proyecto en el que mas horas he invertido y en el que mas cosas diferentes he implementado, desde la gestion de una base y realizar expediciones para conseguir materiales para fabricar objetos y mejoras para la base, hasta un sistema de combate por turnos con similitudes con pokemon pero con mecanicas nuevas.

Interfaz de crafteo - Unity

Una interfaz para craftear objetos, que programe con ScriptableObjects para que se pudiesen añadir mas recetas de forma sencilla.

Escenario procedural 2D y base de datos FireBase - Unity

Este proyecto comenzó con la idea de crear de forma procedural el escenario para un juego de móviles 2D. Basado en parte en el famoso Plantas VS Zombies, diseñe un juego donde coleccionabas criaturas y las podías entrenar jugando partidas. Tiene una base de datos de Firebase que guarda los datos del usuario, los niveles desbloqueados, sus criaturas con todas sus estadísticas y las monedas del juego.
